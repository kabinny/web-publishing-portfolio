Samwon Paper

다양한 종이를 생산하는 삼원특수지 홈페이지의 메인페이지와 소개페이지를 제작해보았습니다. 고딕계열의 두가지 서체만 사용해서 깔끔한 느낌을 주었습니다.
포토샵으로 프로토타입을 만들어보고, 그것을 바탕으로 HTML과 CSS를 이용한 정적인 페이지를 구현해 보았습니다.

Workflow

  • 자료조사와 구상을 하고,
    와이어프레임을 작성

  • 스타일 가이드와
    스토리보드 제작

  • 포토샵을 이용해
    프로토타입 제작

  • HTML / CSS로
    페이지 제작

Wireframe

사이트 분석
1. 종이를 만드는 회사이므로 기본적인 하얀색 바탕에 깔끔한 서체를 사용했습니다.
2. 다양한 종이와 제품을 생산하기 때문에 컬러풀한 이미지를 자주 사용했습니다.

사이트의 정보 구조
비선형적 구조로, 정보가 순차적으로 제공되지 않고 복잡한 네트워크 형식으로 상호 연결되어 사용자가 원하는 페이지로 이동하는 것이 자유로운 구조입니다.

Style Guide

전체적으로 깔끔한 하얀색 바탕에 부분적으로 회색톤 바탕을 사용. 웹사이트 메인 로고는 사이트의 통일감을 주기 위해 기존 CI에서 색을 전부 검정으로 바꾼 것으로 적용했습니다.

폰트는 깔끔하고 세련된 느낌이 들 수 있게 한글에는 노토산스 폰트, 영문에는 몽세라뜨 폰트를 사용했습니다.
